# Basic Graphics Built-in Library - Turtle


**Turtle** module 
- is pre-installed
- simple **graphics** library
- used to create **drawings** and animations
- widely used as **educational tool** for beginners to learn ***basic programming*** concepts in a visual and interactive way.
- requires tkinter to be installed

## Check if tkinter is installed or not

`python3 -m tkinter`

### Install tkinter on Linux (Ubuntu)

`sudo apt install python3-tk`
