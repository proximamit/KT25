# Keyboard Shortcuts for VS Code

| Action                | Shortcut (Windows/Linux)  | Shortcut (macOS)       |
|:---------------------:|:-------------------------:|:----------------------:|
| Open any file by name | `Ctrl + P`                | `Command + P`          |
| Indent a block of code| `Ctrl + ]`                | `Command + ]`          |
| Unindent a code block | `Ctrl + [`                | `Command + [`          |
| Toggle File Explorer  | `Ctrl + B`                | `Command + B`          |
| Select current line   | `Ctrl + L`                | `Command + L`          |
| Delete current line   | `Ctrl + Shift + K`        | `Command + Shift + K`  |
| Find in File          | `Ctrl + F`                | `Command + F`          |
| Find in Project       | `Ctrl + Shift + F`        | `Command + Shift + F`  |
| Find and Replace      | `Ctrl + H`                | `Option + Command + F` |
| Global Replace        | `Ctrl + Shift + H`        | `Shift + Command + H`  |
| Show autocomplete     | `Ctrl + Space`            | `Command + Space`      |

---

### Jump to Function, class or variable definition

- **F12**

> Go to Definition

---

### Find References

- **Shift + F12**

> See where a function or variable is used

---

### Rename a function or variable everywhere

- **F2**

---
### Fold current region

- `Ctrl + Shift + [`

---
### Unfold (Expand) current region

- `Ctrl + Shift + ]`

---


### Fold All (Collapse all regions)

- **Ctrl + K** followed by **Ctrl + 0 (zero)**

---
### Unfold All (Expand all regions)

- **Ctrl + K** followed by **Ctrl + J**

---


## View all shortcuts

- In the command palette, search `open keyboard shortcuts`

or

**Windows/Linux**

- From the Menu (on the top) >> File >> Preferences >> Keyboard Shortcuts

---

**macOS**

- Code >> Preferences >> Keyboard Shortcuts
